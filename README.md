## Weather data API

 A Spring Boot REST API that provides the details of current weather conditions

	
## How to run the app?
	
1 IDE 

	- Run As -> Spring boot app
	
2 Command line

	- $ mvn clean package spring-boot:repackage
	- $ java -jar target/weather-1.0.0.jar
